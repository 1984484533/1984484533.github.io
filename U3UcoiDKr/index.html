个人网盘临时解决方案

容器=网盘？FREE? SPEED?
YES！ YES！ YES！

过去某度网盘一家独大的地位显然已经随着阿里系网盘的出现产生动摇
但作为资深白嫖玩家,Teambition内测赠送的2T容量显然是远远不够的...

github开源的一个项目很自然地提供了无成本高速网盘的解决方案

DockerRegisterCloud

关于DockerRegisterCloud

项目编写仅基于 Docker Registry HTTP API V2 
用户存储内容的隐私性和安全性由用户自己负责以及仓库服务提供商保证

基本说明

支持在浏览器中实现下载而无需客户端
GUI 客户端，预计包括 Linux, Windows, Mac OS, Android, iOS

支持仓库

服务提供商 无需成本 直接下载支持

步骤记录

  1. 阿里云
    1. 注册并登录阿里云
    2. 访问阿里云容器镜像服务，并开通服务
    3. 设置服务密码
    4. 创建私有镜像仓库，代码源选择本地仓库
    5. 复制仓库公网地址，粘贴至仓库地址栏，Enter
    6. 登录阿里云Docker Registry[1]
    7. congratulation!
    8. 注:目前阿里云下载的文件名称都是data，需要自己手动修改文件名以及后缀！
  2. 华为云
    1. 注册并登录华为云
    2. 访问华为云容器镜像服务
    3. 创建组织
    4. 单击登录指令[2]
    5. 真正的地址为:仓库主要地址/组织名称/仓库名称
    6. 仓库名称随意
    7. 地址粘贴至仓库地址栏，Enter
    8. 登录华为云Docker Registry
    9. congratulation!
    10. 注:华为云针对登陆指令做了过期时间的限制，有效期只有1天！

  1. 用于登录的用户名为阿里云账号全名，密码为服务密码可以在访问凭证页面修改凭证密码 ↩︎

  2. -u 后面为用户名;-p 后面为密码;swr....com 为仓库主要地址 ↩︎
